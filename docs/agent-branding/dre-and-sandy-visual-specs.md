# Dre & Sandy — AI Agent Visual Brand Specifications

## Platform Brand System (Voodoo Agents)

| Token | Hex | Usage |
|---|---|---|
| Background | `#050510` | Page background, card backdrops |
| Purple | `#8B31A0` | Primary accent, CTAs, agent tags |
| Gold | `#F5C642` | Secondary accent, highlights, badges |
| Surface | `#0D0D1A` | Cards, modals, glassmorphism bases |
| Border | `rgba(255,255,255,0.08)` | Subtle card/divider borders |

**Typography:** Outfit (Google Fonts) — geometric, modern, all-caps for headings
**Aesthetic:** Glassmorphism + cyberpunk/neon, dark theme, rounded corners (2xl–3xl), backdrop-blur, subtle borders
**Image Style:** Semi-realistic, stylized character avatars, neon-lit, high contrast

## PROMPT GENERATION TIPS

1. **Add `--seed <number>`** to prompts for reproducible results across FLUX runs
2. **Use `--v 2`** for the latest model version
3. **Generate at 2x resolution** and downscale — avoids upscaling artifacts
4. **Generate 4-8 variations per prompt** and select the best
5. **For UI mockups:** Consider generating backgrounds + real UI overlay in Figma for pixel accuracy
6. **For avatars:** Use `--style raw` in FLUX to avoid over-stylization
7. **Consistency trick:** Keep the same character description across all prompts for each agent to maintain visual consistency
